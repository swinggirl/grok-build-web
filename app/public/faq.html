FAQ

What is this?

This is a vibe-coded interface for using Grok Build's video generation function in your browser. It's intended for those who do not want to deal with the terminal for any reason, and for fun.

How does login work?

When you click "Login with Grok," you are redirected to xAI's official authentication servers. You log in securely there, and xAI provides us with a temporary token that allows the site to generate videos on your behalf. We use the exact same OpenID Connect (OIDC) flow that the official Grok Build CLI uses.

Do you store my password or API keys?

No. Because you authenticate directly on xAI's domain, we never see your password. We do not use or require API keys. We only store the access token xAI gives us in a local, private database to keep you logged in across sessions.

Why am I waiting in line?

I'm running this on my home server using consumer fiber internet that I also use for many other things. To keep everything running smoothly for everyone, only 15 people are allowed to be generating videos at any given time.

Can I host this myself?

Yes. I encourage it! The full source code is available via the "Source" link in the header.
