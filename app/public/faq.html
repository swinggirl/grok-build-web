FAQ

What is this?

This is a browser interface for Grok Build's video generation. It is designed to feel like using the official Grok Build CLI without needing the terminal.

How does login work?

When you click "Login with Grok", you authenticate directly with xAI using the same OpenID Connect flow that the official Grok Build CLI uses. Your tokens are stored only in your browser — they never touch our server.

Do you store my tokens or API keys?

No. We do not store access tokens, refresh tokens, or API keys on the server. All authentication happens in your browser. This means we cannot access your tokens.

Why am I waiting in line?

I'm running this on a home server with consumer internet shared with other services. To keep things stable, only 15 people can generate videos at the same time.

Can I host this myself?

Yes. The full source code is available on GitHub.
